# Command Pattern Example - Home Automation System

## Aim
To implement Command Design Pattern to control a Light device.

## Concept
Command Pattern encapsulates a request as an object and allows execution using invoker.

## Files
- Command.java
- Light.java
- LightOnCommand.java
- LightOffCommand.java
- RemoteControl.java
- Main.java

## How to Run
1. Open folder in VS Code
2. Open terminal
3. Compile: javac *.java
4. Run: java Main
