# MVC Pattern Example - Student Management System

## Aim
To implement MVC design pattern for managing student records.

## Components
- Model: Student.java
- View: StudentView.java
- Controller: StudentController.java

## Flow
Controller updates model → View displays updated data

## How to Run
1. Open folder in VS Code
2. Open terminal
3. Compile: javac *.java
4. Run: java Main
