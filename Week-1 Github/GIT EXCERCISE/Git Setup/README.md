# Git Setup

## Objective

Configure Git with user details and verify the configuration.

## Commands Used

```bash
git config --global user.name "YourName"
git config --global user.email "yourmail@gmail.com"
git config --list
```

## Explanation

* `git config --global user.name` sets the Git username.
* `git config --global user.email` sets the Git email address.
* `git config --list` displays the current Git configuration.

## Result

Git was configured successfully with the user details.
