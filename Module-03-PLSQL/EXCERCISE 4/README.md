# Exercise 4: Functions

## Scenario 1: CalculateAge

### Objective

Calculate the age of a customer using the date of birth.

### Logic

* Accept date of birth as input.
* Calculate the difference between current date and date of birth.
* Return age in years.

### Result

The function successfully returns the customer's age.

---

## Scenario 2: CalculateMonthlyInstallment

### Objective

Calculate the monthly installment amount for a loan.

### Logic

* Accept loan amount, interest rate, and duration.
* Compute total payable amount.
* Divide by total number of months.
* Return monthly installment value.

### Result

The function successfully calculates the monthly installment amount.

---

## Scenario 3: HasSufficientBalance

### Objective

Check whether an account has enough balance for a transaction.

### Logic

* Retrieve account balance.
* Compare balance with requested amount.
* Return TRUE if sufficient, otherwise FALSE.

### Result

The function successfully validates account balance before transactions.

---

## Conclusion

This exercise demonstrates the creation and usage of PL/SQL functions for age calculation, loan installment computation, and balance validation.
