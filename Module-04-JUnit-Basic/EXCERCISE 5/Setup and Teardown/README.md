# Exercise 5: Setup and Teardown Methods

## Aim

To demonstrate setup and teardown methods used before and after test execution.

## Description

Setup initializes the required objects before running the test. Teardown performs cleanup activities after test execution.

## Program

A BankAccount object is initialized during setup, tested for balance verification, and cleaned up during teardown.

Thus, the setup and teardown process was implemented and verified successfully.
