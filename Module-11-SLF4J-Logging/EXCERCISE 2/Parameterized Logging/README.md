# Exercise 2: Parameterized Logging

## Task
Write a Java application that demonstrates parameterized logging using SLF4J.

## Step-by-Step Solution

1. Added SLF4J and Logback dependencies to the `pom.xml` file.
2. Created a Java class `ParameterizedLoggingExample` that uses SLF4J for parameterized logging. 
3. Used `{}` placeholders to dynamically insert variables into the log string.

**Note on Screenshots:** Please run `ParameterizedLoggingExample.java` and capture the console output to save as `Screenshot.png` in this directory.
