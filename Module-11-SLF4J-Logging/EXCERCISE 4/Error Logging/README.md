# Exercise 4: Error Logging

## Task
Write a Java application that demonstrates logging error messages using SLF4J.

## Step-by-Step Solution

1. Added SLF4J and Logback dependencies to the `pom.xml` file.
2. Created a Java class `ErrorLoggingExample` that uses SLF4J for logging error messages and exceptions. 
3. Used `logger.error()` to print out errors and stack traces.

**Note on Screenshots:** Please run `ErrorLoggingExample.java` and capture the console output to save as `Screenshot.png` in this directory.
