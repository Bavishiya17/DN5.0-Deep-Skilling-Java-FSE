# Git Ignore

## Objective

Ignore unnecessary files and folders from being tracked by Git.

## .gitignore Content

```text
*.log
*.tmp
*.bak
backup/
bin/
target/
*.class
```

## Explanation

* `*.log` ignores log files.
* `*.tmp` ignores temporary files.
* `*.bak` ignores backup files.
* `backup/` ignores the backup folder.
* `bin/` ignores compiled binaries.
* `target/` ignores build output.
* `*.class` ignores Java class files.

## Result

Git ignores the specified files and directories during tracking.
