# Exercise 4 - Employee Management System

## Array Representation

Arrays store elements in contiguous memory locations.

### Advantages

- Fast access using index.
- Easy to implement.
- Less memory overhead.

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Analysis

Arrays provide efficient index-based access but have fixed size.

Deletion requires shifting elements, making it slower.

Arrays are suitable when the number of records is fixed and random access is required.
