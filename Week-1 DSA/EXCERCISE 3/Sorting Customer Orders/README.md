# Exercise 3 - Sorting Customer Orders

## Sorting Algorithms

- Bubble Sort compares adjacent elements and swaps them.
- Insertion Sort inserts elements into their correct position.
- Quick Sort uses a pivot and partitions the array.
- Merge Sort divides the array and merges sorted parts.

## Time Complexity

| Algorithm | Best | Average | Worst |
|-----------|------|----------|--------|
| Bubble Sort | O(n) | O(n²) | O(n²) |
| Quick Sort | O(n log n) | O(n log n) | O(n²) |

## Analysis

Bubble Sort performs many unnecessary comparisons and is slow for large datasets.

Quick Sort divides the array into smaller parts, making it much faster in practice.

Therefore, Quick Sort is generally preferred for sorting customer orders on an e-commerce platform.
