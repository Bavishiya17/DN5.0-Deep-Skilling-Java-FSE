# Dependency Injection Example - Customer Management System

## Aim
To implement Dependency Injection using constructor injection.

## Concept
CustomerService depends on CustomerRepository and dependency is injected externally.

## Files
- CustomerRepository.java
- CustomerRepositoryImpl.java
- CustomerService.java
- Main.java

## How to Run
1. Open project in VS Code
2. Open terminal
3. Compile: javac *.java
4. Run: java Main
