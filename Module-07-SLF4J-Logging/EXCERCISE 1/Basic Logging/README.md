# Exercise 1: Basic Logging

## Task
Write a Java application that demonstrates basic logging using SLF4J (`info` and `debug` levels).

## Step-by-Step Solution

1. Added SLF4J and Logback dependencies to the `pom.xml` file.
2. Created a Java class `BasicLoggingExample` that uses SLF4J for logging basic informational and debug messages.

**Note on Screenshots:** Please run `BasicLoggingExample.java` and capture the console output to save as `Screenshot.png` in this directory.
