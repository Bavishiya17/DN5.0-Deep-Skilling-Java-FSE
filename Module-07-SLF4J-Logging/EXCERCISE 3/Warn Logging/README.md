# Exercise 3: Warn Logging

## Task
Write a Java application that demonstrates logging warning messages using SLF4J.

## Step-by-Step Solution

1. Added SLF4J and Logback dependencies to the `pom.xml` file.
2. Created a Java class `WarnLoggingExample` that uses SLF4J for warning logging. 
3. Used `logger.warn()` to print out warning messages.

**Note on Screenshots:** Please run `WarnLoggingExample.java` and capture the console output to save as `Screenshot.png` in this directory.
