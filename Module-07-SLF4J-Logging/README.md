# Module 7: SLF4J LOGGING

This module contains exercises to demonstrate logging using SLF4J (Simple Logging Facade for Java) backed by Logback.

## Topics Covered
1. Basic Logging (`logger.info`, `logger.debug`)
2. Parameterized Logging (Using `{}` placeholders)
3. Warn Logging (`logger.warn`)
4. Error Logging (`logger.error` and exception stack traces)
5. Appenders (Configuring `ConsoleAppender` and `FileAppender` in `logback.xml`)

## Structure
Each exercise is provided in its respective directory (`EXCERCISE 1` through `EXCERCISE 5`) containing:
- `README.md`
- `Source Files` folder with the Java source and any required configuration files.
- `Screenshot.png` (To be added manually after running the code)

You can also run all the examples from the root `src/main/java/com/cognizant/slf4j/` folder via Maven or your IDE.
