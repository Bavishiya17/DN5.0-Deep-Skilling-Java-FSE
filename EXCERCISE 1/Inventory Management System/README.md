# Inventory Management System

## Algorithm
1. Create a HashMap to store products.
2. Add products using product ID as key.
3. Update product details using HashMap.get().
4. Delete products using HashMap.remove().
5. Display all products.

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(1) |
| Update | O(1) |
| Delete | O(1) |
| Search | O(1) |

## Result

Inventory operations are performed efficiently using HashMap, making the system suitable for handling large inventories.
