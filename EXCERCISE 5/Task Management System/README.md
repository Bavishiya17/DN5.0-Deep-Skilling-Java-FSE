# Exercise 5 - Task Management System

## Linked Lists

### Singly Linked List
Each node contains data and a reference to the next node.

### Doubly Linked List
Each node contains data, previous node reference, and next node reference.

## Time Complexity

| Operation | Complexity |
|-----------|------------|
| Add | O(n) |
| Search | O(n) |
| Traverse | O(n) |
| Delete | O(n) |

## Analysis

Linked lists allow dynamic memory allocation and easy insertion or deletion without shifting elements.

They are preferred over arrays when the size of data changes frequently.
